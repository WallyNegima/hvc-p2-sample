----------------------------------------------------
 B5T-007001 サンプルコード
----------------------------------------------------

(1) 本提供物について
  B5T-007001(HVC-P2)のサンプルコードを提供いたします。
    1-1) B5T-007001とコマンド送受信(機能実行)を行うサンプルコード
    1-2) HVCのセンシング結果を安定化する結果安定化ライブラリ(STBLib)のソースコード
    1-3) STBLibを利用して、B5T-007001のセンシング結果を安定化させるサンプルコード

(2) ディレクトリ構成
      SampleCode/           B5T-007001とコマンド送受信(機能実行)を行うサンプルコード           ・・・(上記 1-1)
         bin/               ビルド時の出力ディレクトリ
         platform/          VC10/VC14でのビルド環境
         src/               ソースコード本体
      STBLib/               STBLib関連の一式                                                   ・・・(上記 1-2)
         lib/               STBLib本体
            doc/            STBLibに関する資料一式
            bin/            STBLib  バイナリファイル本体
            platform/       VC10(Visual Studio 2010 C++)/VC14(Visual Studio 2015 C++)でのビルド環境
            src/            STBLib  ソースコード本体
         SampleCode/        B5T-007001とコマンドを送受信し、結果安定化を実施するサンプルコード ・・・(上記 1-3)
            bin/            ビルド時の出力ディレクトリ
            import/         STBLibを利用するためのインポートディレクトリ
            platform/       VC10/VC14でのビルド環境
            src/            ソースコード本体

* STBLibをVC10/VC14にてビルドする際はMFCライブラリが必要です。事前にインストールしてください。
* 各提供物の詳細は、同梱された資料をご確認ください。
